# Images Folder

This folder contains all images and visual assets for the SRTTC Leave & Bill Management System.

## Required Images

### 1. SRTTC Logo (srttc-logo.png)
- **File**: The SRTTC (Sundar Ramasamy Tulasiani Technical Campus) official logo
- **Format**: PNG with transparent background preferred
- **Dimensions**: Minimum 500x500px (will be scaled as needed)
- **Usage**: Header, landing page, favicon source

### 2. Favicon (favicon.ico)
- **File**: Icon version of SRTTC logo
- **Format**: .ico format
- **Dimensions**: 32x32px and 16x16px
- **Usage**: Browser tab icon

### 3. Additional Assets
- `default-avatar.png` - Default user profile picture
- `placeholder.png` - General placeholder image

## Image Upload Instructions

1. Save the SRTTC logo image provided
2. Upload via GitHub: 
   - Click "Upload files" button
   - Drag and drop the logo file
   - Name it exactly: `srttc-logo.png`
3. Convert logo to favicon.ico (use online tool like favicon.io)
4. Upload favicon.ico to this folder

## Image Guidelines

- All images should be optimized for web (compressed)
- Use PNG format for logos and icons (transparency support)
- Use JPG for photographs
- Maximum file size: 500KB per image
- Name files using lowercase with hyphens (kebab-case)
